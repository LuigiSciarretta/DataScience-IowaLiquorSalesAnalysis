Il progetto è relativo ad uno dei 4 progetti svolti durante il corso di Data Science presso l'università Politecnica delle Marche, nell'anno 2019.

L'obiettivo è stato comprendere ed utilizzare il linguaggio Python con l'ecosistema di librerie per la Data Science. 
Per il progetto, si è scelto il Dataset Iowa Liquor Sales, rappresentante i dati delle vendite all'ingrosso di superalcolici in America.

Sono state condotte diverse analisi descrittive per comprendere gli eventi passati e identificare i principali venditori, i marchi più popolari e, di conseguenza, le preferenze degli acquirenti nel settore degli alcolici. Inoltre, è stata esplorata la relazione tra diverse variabili, come ad esempio il rapporto tra il numero di bottiglie vendute e il loro prezzo, nonché la correlazione tra la popolazione di determinate aree e i profitti generati.

Per affrontare queste analisi, sono state impiegate librerie di Machine Learning al fine di ottenere previsioni. In particolare, attraverso il cosiddetto "Spatial Data Mining", sono state individuate le zone dell'Iowa in cui potrebbe essere vantaggioso aprire nuovi negozi di liquori.

Sempre attraverso l'utilizzo delle librerie di Machine Learning, sono state esaminate le vendite dei principali marchi concorrenti, trattandole come serie temporali. Di conseguenza, sono state condotte analisi temporali per prevedere le vendite future dei marchi e stimare l'andamento futuro del mercato americano dei liquori.

Il file "project-documentation" è una tesina del progetto, in cui sono racchiuse i dettagli di analisi.
